# @hpcc-js/chart

This package contains @hpcc-js bespoke "charts".  It contains the following widgets:
* [Area](./Area.md)
* [Bar](./Bar.md)
* Bubble
* BubbleXY
* Bullet
* [Column](./Column.md)
* Contour
* Gantt
* Gauge
* [HalfPie](./HalfPie.md)
* HexBin
* Line
* [Pie](./Pie.md)
* [QuarterPie](./QuarterPie.md)
* Radar
* RadialBar
* Scatter
* Summary
* WordCloud
